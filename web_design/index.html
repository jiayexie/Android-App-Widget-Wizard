INTRO USAGE

If you have an Android phone or tablet, you will be attracted to the convenience and powerfulness of Android App Widget.

But sometimes it's hard to find the very Widget you need, and sometimes you just don't like the way existing Widgets are.

The AAWW project aims at solving these problems.

AAWW (Android App Widget Wizard) is designed to provide Android users with a simple way to DIY Android App Widgets.

On the AAWW web platform, you can easily create your favorite application shortcuts, browser bookmarks, fast dial, etc, or a combination of them.

Here is the instruction of AAWW:

  1. Visit http://162.105.81.63/web_design
  2. Input the name of the Widget, size of the Widget, and finally your name. And press "START!"
  3. Design the Layout in the workspace, creating new components from the component library
  4. Adjust the position and size of the component with your mouse, and edit other attributes of the component with your keyboard
  5. Upload some resource pictures to customize the appearance of the widget
  6. Edit the function of each component and check the logic of each component
  7. Confirm everything, and you can sumbit your widget configuration to the server
  8. Wait for the apk file to download; install it on your Android device.
  9. Enjoy!
